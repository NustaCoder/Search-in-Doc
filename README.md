# Search-in-Doc
Search-in-Doc

## Goal
This application enables intelligent search, question answering, and summarization across user-provided documents using AI agents.

## Features
- Retrieval-Augmented Generation (RAG) pipeline for enhanced document search and Q&A
- Document storage and retrieval powered by ChromaDB
- Local artifact saving for future reference
- AI agent workflow for document analysis and context-based answers
